# Nodejs joke bot

Nodejs-joke-bot shows Chuck Norris jokes from http://www.icndb.com/api/

### Pre requirement
   -   Nodejs
   -   npm

### Install and Start
 - `git clone https://github.com/macroramesh6/nodejs-joke-bot.git`
 - `cd nodejs-joke-bot`
 - `npm install`
 - `npm start`

### Start conversation
Start your conversation with `hi` or `hello` or `help` etc.,
